# 👽 외계인
처음에 사용할 수 있는 캐릭터

## 성격
항상 엉뚱한 행동을 한다
뭘 할지 아무도 모른다

## 기본 공격
🟢 이상한 레이저 빔

## 스킬
### ✨ 순간이동
짧은 거리로 갑자기 이동

### 🧠 텔레파시
적의 체력을 조금 뺏어서 자기 체력으로 회복

### 👁️ 초능력
숨어 있는 적을 찾아낸다
발견된 적을 자동으로 공격하거나 표시해 준다

## 👉 외계인 요약:
도망 잘 침 + 체력 뺏음 + 숨은 적 잡는 탐색 캐릭터

# 👀 눈 돌아가는 사람
두번째로 해금되는 캐릭터

## 성격
눈이 돌아가 있고
주변을 헷갈리게 만든다

## 기본 공격
### 🌫️ 이상한 파동
데미지
혼란 효과
혼란에 걸리면 조작이 반대로 움직인다

## 스킬
### 🌀 광기 산책 (이름은 나중에 바꿔도 됨)
스킬을 켜면 움직일 때마다 근처에 있는 적의 체력이 조금씩 깎인다

# 🍛 카레 곰돌이
눈 돌아가는 사람보다 더 나중에 해금되고 더 강함
잘 안 죽는 탱커

# 기본 공격
## 🍲 냄비 내려치기
앞에 있는 적을 강하게 때린다
덤으로 카레가 흩뿌려진다
맞은 적 주변에도 살짝 피해

## 패시브(특징)
맞으면 받은 대미지의 절반을 저장
체력이 바로 줄어들지만 나중에 다시 돌려받을 수 있다

# 스킬
## 🍛 카레 회복
버튼을 누르면 저장해 둔 대미지를 체력으로 회복

# 💪 헐크 언니 (완성)
카레 곰돌이보다 더 강함
오래 살수록 가장 위험한 캐릭터 중 하나

## 기본 스탯
❤️ 시작 체력이 높다

## 패시브
😤 피해를 받을 때마다 점점 더 강해진다
(공격력 / 던지기 세기 증가)

## 기본 공격
### 🌀 적을 들어서 돌려 던지기
가까운 적을 붙잡아 던진다
던져진 적은 충돌 피해

## 스킬
### 🔥 폭주
버튼을 누르면 잠깐 동안 엄청 세진다
공격력, 던지기 거리 크게 증가

# 🐭 찍찍찍찍찍
## ❗ 핵심 규칙
❌ 기본 공격이 없다
🛠️ ‘만들기’만 가능하다
📦 만든 것은 최대 3개까지 저장
🎮 만든 걸 사용하는 전용 스킬이 있다

## 🛠️ 만들기 (기본 행동)
버튼을 누르면 찍찍 관련 물건 / 가짜 / 장치를 만든다
만든 것은 바로 공격하지 않음
인벤토리처럼 쌓아둠 (최대 3개)

## 🎮 사용 스킬
쌓아둔 것을 한 번에 하나씩 사용
### 사용하면:
가짜가 튀어나오거나
적을 헷갈리게 하거나
수면 가루가 터지거나
쥐 인형으로 변하게 만들거나
즉, 설치 + 트릭형 캐릭터

### 😴 수면 가루 (확정 스킬)
만든 것 중 하나
사용하면 적을 잠들게 함
그 사이에 팀원이 공격 가능

# 🧑‍🏫 선생님
## 🎭 위장 능력
선생님은 다른 캐릭터로 위장할 수 있다
위장 시 겉모습만 변경된다
처음에는 아무도 선생님인지 알 수 없다

## 🎮 위장 상태 규칙
위장한 상태에서도: 기본 공격 1개 사용 가능
스킬 1개 사용 가능
완전히 무력한 상태 ❌
들키지 않으면 일반 캐릭터처럼 행동 가능

### ⏳ 위장 제한
위장은 1분 동안 유지
1분이 지나면: 정체가 자동으로 드러난다
위장이 풀린다

### 📈 레벨업 보상
선생님이 레벨업하면:
다시 랜덤하게 다른 캐릭터로 위장
⏱️ 위장 유지 시간이 더 길어진다
