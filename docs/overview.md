# 🎮 게임 이름
미친 제자들 (Unhinged Students)
(웹 게임)

## 🧒 이 게임은 어떤 게임이야?
- 여러 사람이 같이 하는 팀 게임
- 레드팀 vs 블루팀 **5 : 5 팀전**
- 플레이어는 **캐릭터 한 명만 조종**
- 실시간 멀티플레이

## 🧭 전체 스크린 플로우 요약
접속
→ 접속 보상
→ 로비
→ 캐릭터 관리
→ 매치 시작
→ 게임 플레이
→ 사망 / 리스폰
→ 게임 종료
→ 결과 & 보상
→ 로비

## 1️⃣ 접속 보상 화면 (Daily / Weekly)

### 화면 역할
- 게임에 처음 접속했을 때 자동으로 등장
- 매일 받을 수 있는 **랜덤 캐릭터 아이템 제공**
- 매일 접속하면 랜덤으로 캐릭터 아이템을 얻을 수 있다.
- 아이템은 캐릭터 전용이다.
- 아이템이 나올 때 빛이 번쩍 나오고 아이템 그림이 크게 나온다.
- 글자는 아이템 이름만 나온다.

### 화면 구성
- DAY 버튼
- 아이템 미리보기
- 보상 획득 버튼

### 동작 흐름
- DAY 버튼 클릭
- 아이템 획득 연출 화면으로 이동

## 2️⃣ 아이템 획득 연출 화면 (Overlay)

### 화면 역할
- 아이템을 얻었을 때 보여주는 연출 화면

### 연출 요소
- 화면 어두워짐
- ✨ 빛이 번쩍
- 아이템 이미지 크게 표시
- 아이템 이름만 표시
- 닫기 버튼

### 다음 화면
- 닫기 → 로비 화면

## 3️⃣ 로비 화면 (메인 허브)

### 화면 역할
- 게임의 중심 화면
- 모든 다른 화면으로 이동 가능

### 가능 행동
- ▶️ 게임 시작
- 👤 캐릭터 화면 이동
- 💎 재화 확인 (루비 / 다이아)
- 📅 접속 보상 확인

## 4️⃣ 캐릭터 화면 (캐릭터 & 아이템 관리)

### 화면 역할
- 캐릭터 관리 및 성장 확인

### 가능 행동
- 캐릭터 선택
- 레벨 확인
- 아이템 장착 / 해제
- 숫자 상승 확인 (레벨, 수치)

### 핵심 포인트
- 외형보다 **숫자 상승이 중요**

### 이동
- 뒤로 → 로비
- 게임 시작 → 매치 시작 화면

## 5️⃣ 매치 시작 화면 (팀 배정)

### 화면 역할
- 게임 시작 전 준비 화면

### 화면 구성
- 레드팀 / 블루팀 자동 배정
- 팀별 캐릭터 5명 표시
- 곧 시작된다는 안내

### 다음 화면
- 자동으로 게임 플레이 화면 진입

## 6️⃣ 게임 플레이 화면 (전투)

### 화면 역할
- 실제 게임이 진행되는 메인 화면
- 게임이 시작되면 여러 캐릭터가 같이 나온다.
- 레드팀과 블루팀은 자동으로 정해진다.
- 바닥에는 샤드와 다이아가 떨어져 있다.

### 💎 샤드와 레벨
샤드를 주우면 캐릭터가 커지지는 않고 레벨 숫자만 올라간다.
레벨이 올라가면 더 강해진다.

### ⚔️ 전투
다른 팀 캐릭터를 버튼을 눌러 공격한다.
리그 오브 레전드처럼 싸운다.
죽으면 잠깐 기다렸다가 다시 나온다.
죽을수록 다시 나오는데 시간이 더 오래 걸린다.

### 플레이 요소
- 캐릭터 이동
- 공격 버튼
- 스킬 사용
- 샤드 / 다이아 획득
- 레벨 업
- 팀 전투

### UI 요소
- 체력
- 레벨
- 공격 버튼
- 스킬 버튼
- 킬 수

## 7️⃣ 사망 화면 (Death / Respawn)

### 화면 역할
- 캐릭터 사망 시 표시

### 표시 정보
- 내가 죽인 적 수
- 현재 사망 순위
- 리스폰까지 남은 시간

### 특징
- 죽을수록 리스폰 시간이 증가
- 게임은 계속 진행됨

### 다음 화면
- 리스폰 → 게임 플레이 화면

## 8️⃣ 게임 종료 / 결과 화면

### 화면 역할
- 게임 종료 후 결과 표시

### 표시 내용
- 승리 팀
- 내 킬 수
- 팀 결과 요약

### 다음 화면
- 결과 확인 → 보상 화면

## 9️⃣ 보상 정산 화면

### 화면 역할
- 게임 결과에 따른 보상 지급
- 한 판은 정해진 시간이 지나면 끝난다.
- 시간이 끝났을 때 상대를 더 많이 죽인 팀이 이긴다.

### 보상 내용
- 획득 루비 (킬 수 기준)
- 다이아 변화 (획득 / 손실)
- 아이템 획득 여부

### 다음 화면
- 확인 → 로비 화면

## 💰 재화 시스템 요약

### 루비
- 게임 종료 후 킬 수만큼 획득
- 아이템 잠금 해제

### 다이아
- 게임 중 바닥에 떨어진 것을 획득
- 사망 시 일부 잃음
- 캐릭터 강화
- 캐릭터 해제
